# Plugins

If the version of a program matters a lot, I use ASDF. Here are a list of plugins I use:

## yq

```
asdf plugin add yq https://github.com/sudermanjr/asdf-yq.git
```

## kubectl

```
asdf plugin add kubectl https://github.com/asdf-community/asdf-kubectl.git
```

## OpenTofu

```
asdf plugin add opentofu https://github.com/virtualroot/asdf-opentofu.git
```

## kustomize

```
asdf plugin add kustomize https://github.com/Banno/asdf-kustomize.git
```
